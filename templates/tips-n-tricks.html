Customize the size

The easiest way to set the size of the icon is to add width and height attribute to the SVG tag. Another way is to wrap the icon inside an HTML element with specified width and height.

Customize the color

You can customize the color of the icons with the fill and stroke property. And it works for hovering too, for example: 

<div class="icon">
  <svg>
    <use xlink:href="font-awesome-solid.svg#heart"/>
  </svg>
</div>

.icon-heart {
  fill: red;
}

.icon-heart:hover {
  fill: orange;
}
